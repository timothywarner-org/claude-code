# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please contact Tim Warner at [tim@techtrainertim.com](mailto:tim@techtrainertim.com) or visit [TechTrainerTim.com](https://techtrainertim.com).

We take all security reports seriously and will respond as quickly as possible.

## Supported Versions

Only the latest version of this repository is supported with security updates.

## Responsible Disclosure

Please do not publicly disclose vulnerabilities until they have been reviewed and addressed by the maintainers.
